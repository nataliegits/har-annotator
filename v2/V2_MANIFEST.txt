HAR-annotator v2.0 bundle
=========================
Drop-in additions for github.com/nataliegits/har-annotator. Nothing here
overwrites v1 — the v2 files are named distinctly (README_v2.md,
demo_live_v2.py, discovery/neglect shortlists) so the disease-anchored v1
pipeline stays intact for reference.

v2.0 = discovery mode: same pipeline, disease-gene gate dropped.
       363 disease-anchored -> 577 (363 shared + 214 newly surfaced near
       non-disease genes). Strict superset of v1.
v1.1 = neglect-aware ranking: optional axis that rewards understudied
       target genes (bundled here as it shipped alongside v2).

CONTENTS
--------
README_v2.md                       v2 README (discovery mode, no disease requirement)
demo_live_v2.py                    live demo, superset of v1's demo_live.py
                                   (adds --discovery and --neglect W)

har_shortlist_discovery.parquet    577-element discovery shortlist (full)
har_shortlist_discovery.csv          "        "         "        (CSV)
discovery_new_elements.csv         the 214 newly-surfaced elements only

har_shortlist_neglect.parquet      neglect-aware re-ranking (v1.1)
har_shortlist_neglect.csv            "        "        "     (CSV)

validation_ladder.md               wet-lab validation path (MPRA -> Capture-C
                                   + CRISPRi -> humanized organoids)

figures/fig_discovery_vs_anchored.png   funnel + score dists + top-14 new genes
figures/fig_neglect_rerank.png          rank reshuffle under neglect weighting
figures/fig_validation_ladder.png       the 4-tier validation schematic

data/har_ortholog_seqs_relaxed.parquet  panTro5 ortholog cache for the 214 new
                                        elements (needed to reproduce v2 scores)

HOW TO RUN
----------
conda activate hars
python demo_live_v2.py --discovery        # the 577-element shortlist, live
python demo_live_v2.py --neglect 0.25     # neglect-aware re-ranking
(v1 unchanged: python demo_live.py / python run_pipeline.py)
