HAR-annotator v2.0 bundle
=========================
Drop-in additions for github.com/nataliegits/har-annotator. Nothing here
overwrites v1 — the v2 files are named distinctly (README_v2.md,
demo_live_v2.py, discovery/relaxed/neglect shortlists) so the disease-anchored
v1 pipeline stays intact for reference.

THREE NESTED RELAXATION LEVELS (each removes one gate; 363 ⊂ 577 ⊂ 2,757):
  v1  disease-anchored   363   constraint + GWAS + near a DDG2P disease gene
  v2  discovery          577   constraint + GWAS + near ANY gene (disease gate dropped)
      constraint-only  2,757   constraint only (GWAS gate also dropped; 5-axis score)

v1.1 = neglect-aware ranking: optional axis that rewards understudied
       target genes (bundled here as it shipped alongside v2).

HOW TO RUN
----------
conda activate hars
python demo_live_v2.py --discovery        # the 577-element shortlist, live
python demo_live_v2.py --neglect 0.25     # neglect-aware re-ranking
# constraint-only (2,757): load har_shortlist_relaxed.parquet and re-rank
#   (precomputed table, no demo flag — see README_v2 "three relaxation levels")
(v1 unchanged: python demo_live.py / python run_pipeline.py)
